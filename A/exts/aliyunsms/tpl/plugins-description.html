阿里云短信登录注册插件

作者：留恋风2581047041@qq.com  

插件功能
使用阿里云短信，实现发送手机号验证码，使用手机号注册，登录等操作。
如何使用？
1. 需要加入相关代码实现手机号发送及验证
2. 配置短信相关信息
3. 每次发送短信验证码的时候必须先写图形验证码，这个是安全规范
4. 下面是添加的前端代码：
login.html页面
			  

				
				
				
			  

			function sendmsg(){
				var tel = $("#tel").val();
				if(tel==''){
					alert('手机号不能为空！');
					return false;
				}
				var vercode = $("#yzm").val();
				if(vercode==''){
					alert('图形验证码不能为空！');
					return false;
				}
				
				$.post('/msg/index',{tel:tel,yzm:vercode,codename:'login_vercode'},function(r){
					alert(r.msg);
				
				},'json');
				

			}
			register.html页面
